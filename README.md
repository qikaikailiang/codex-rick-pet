# Codex Rick Pet

一个可直接导入 Codex 的自定义 Pet 示例仓库，角色基于《Rick and Morty》中的 Rick。

仓库内容包括：

- `codex-rick-pet-csdn.md`：完整中文教程
- `assets/tutorial/`：教程配图
- `pet/rick-portal/`：可直接导入 Codex 的 Pet 文件

## 使用方法

1. 将 `pet/rick-portal` 复制到本地 `~/.codex/pets/` 目录下。
2. 打开 Codex。
3. 进入 `Settings -> Appearance -> Pets`。
4. 刷新自定义宠物并选择 `Rick Portal`。

## 仓库结构

```text
.
├── assets/tutorial/
├── codex-rick-pet-csdn.md
├── pet/rick-portal/
│   ├── pet.json
│   └── spritesheet.webp
└── README.md
```

## 说明

本仓库只保留最终可用的 Pet 产物和教程文档，不包含生成过程中的临时目录和中间文件。
